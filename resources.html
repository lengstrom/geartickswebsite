Git Presentation

Our presentation about using Git and GitHub, which revolutionized our programming this year. We use Git to sync our code easily across many computers and to allow multiple programmers to make changes to the code at the same time. GitHub provides a great interface to review changes we have made to the code and to revert to older versions if necessary. Maintaining multiple branches of the code is also useful, as it allows programmers to experiment with various things without having to worry about messing up a working version. Being able to revert changes and to automatically merge changes also makes it much easier for beginners to program. We have asked FIRST to present about Git and GitHub at the World Festival, whether or not our team makes it there.
